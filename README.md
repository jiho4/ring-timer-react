# Ring Timer (react ver.)
*created: 2024/08/15*

## Description

A React app that runs a timer with a bell sound at specified intervals.

## Usage

- Input `interval time` (second)
- Press `space bar` to continue the loop
- Press `esc` to reset

## Build & Run

```
// dev
npm start

// prod
npm install react-scripts@latest
npm run build

npm install -g serve
serve -s build
```
